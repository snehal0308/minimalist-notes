# Flask Web App - Minimalist Notes 

## Setup & Installtion

Make sure you have the latest version of Python installed.

```bash
git clone https://github.com/snehal0308/minimalist-notes.git
```

## Running The App

```bash
python main.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
